Supported Modes

Mode A
On-Policy
staleness=0 sync=1
Produce one batch, train, sync immediately. Maximum freshness.
1.0× baseline
Mode B
Stream Off-Policy
staleness=0 sync>1
Multiple local updates between syncs. Streaming reduces bubbles.
~1.6× speedup
Mode C
Async + Stale
staleness>0 partial=off
Allow stale samples up to threshold. Wait for active tasks on sync.
~2.0× speedup
Mode D
Async + Partial
staleness>0 partial=on
Interrupt in-progress generation on sync. Maximum overlap.
~2.7× speedup
