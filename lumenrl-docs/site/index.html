Key Features

High-performance RL post-training for LLMs on AMD GPUs — Lumen quantized training and ATOM optimized inference

⚡

AMD-Native

AITER kernels, MORI communication, and a ROCm-first stack tuned for AMD Instinct — from fused ops to device-side collectives and MoE expert dispatch.

⚙

FP8 End-to-End

FP8 rollout (W8A8 + KV-cache) plus FP8 training with hybrid formats — up to 44% throughput gain over BF16 on representative workloads.

▲

MoE-Stable RL

R3 (Rollout Routing Replay) records inference routing and replays it in training so train and inference routers stay aligned.

♺

Flexible Backends

FSDP2 or Megatron-Core for policy training with Lumen FP8, plus ATOM for rollout with TP, EP, and DP parallelism.

☷

Multiple Algorithms

GRPO, DAPO, and PPO with clipped surrogate objectives, advantage normalization, and group-relative baselines where applicable.

⚗

Production-Ready

Ray orchestration, YAML-driven configs, multi-node SLURM launch scripts, and Docker images for reproducible clusters.

Architecture

Ray controller, ATOM inference with FP8 rollout and R3 recording, Lumen training with FP8 and R3 replay — AITER kernels and MORI comms end to end

Quick Start

Docker or editable install, then launch GRPO with BF16, FP8, or MoE R3 configs

Install
# Docker (recommended)
docker pull rocm/atom-dev:latest
docker run -it --network=host \
  --device=/dev/kfd --device=/dev/dri \
  --group-add video --cap-add=SYS_PTRACE \
  --security-opt seccomp=unconfined \
  --shm-size=16G -v $HOME:/home/$USER \
  rocm/atom-dev:latest

# Inside container — developer install
git clone https://github.com/ZhangDanyang-AMD/Lumen-RL.git && cd Lumen-RL
pip install -e ".[all]"
Run GRPO
# BF16 baseline (single GPU)
python examples/run_grpo.py --config configs/grpo_dense_bf16.yaml

# FP8 end-to-end
python examples/run_grpo.py --config configs/grpo_dense_fp8.yaml

# 8 GPUs
python examples/run_grpo.py --config configs/grpo_dense_fp8_8gpu.yaml

# MoE + R3 + FP8
python examples/run_grpo.py --config configs/grpo_moe_fp8_r3.yaml
FP8 Config
quantization:
  rollout:
    precision: fp8
    use_deep_gemm: true
    num_first_layers_in_bf16: 0
    num_last_layers_in_bf16: 0
  training:
    fp8: hybrid
    fp8_recipe: blockwise
    fp8_weight_cache: true
  rollout_correction:
    enabled: true
    method: tis
    clip: 1.5
MoE R3 Config
moe:
  r3:
    enabled: true
    record_router_logits: true
    replay_mode: distribution
